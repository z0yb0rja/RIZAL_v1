# RIZAL_v1
**Software Design: Developing an Advance Mobile Attendance System**

Welcome to the **R.I.Z.A.L.** repository! R.I.Z.A.L. stands for **Record & Integrated Zone for Attendance & Logging**. 

<img width="299" height="565" alt="image" src="https://github.com/user-attachments/assets/110aa556-b847-4020-9314-97ac2627e9cd" /> 


This project aims to modernize and streamline attendance tracking by utilizing an advanced, mobile-responsive interface and facial recognition technology. 

## 🚀 Features
* **Secure Login & Authentication:** Role-based access for Students, Student Government (SG), and IT Administrators.
* **Face Verification:** Integrated facial recognition for secure and accurate attendance logging.
* **"Remember Me" Functionality:** Streamlined session management for returning users.
* **Interactive Dashboard:** Clean, user-friendly interface for managing records, approvals, and system settings.

## 💻 Tech Stack
* **Frontend:** Vue.js (located in the `rizal-app` directory)
* **Deployment & Containerization:** Docker (`docker-compose.yml`)
* **Styling:** Tailwind CSS

## 📂 Repository Structure
* `/rizal-app` - Contains all the frontend code, components, and views.
* `docker-compose.yml` - Configuration for running the application environment.
* `*.md` files - Documentation and guides for specific features (like Face Recognition setup).
